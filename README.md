# hide-and-seek
Basic and simple minecraft hide and seek plugin

# Commands
 /has **-** Main command for Hide And Seek Plugin, shows Help Menu
 /has help **-** shows Help Menu
 /has createarena [name] [<world>] ... **-** Creats a new has arena
